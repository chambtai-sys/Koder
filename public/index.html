Quick Start

Interactive Guide: How to Use Spark Language

Spark is a native, lightweight language engineered inside Koder specifically for modeling and rendering **Micro App Dashboards** instantaneously. It lets you define metadata, application logs, and visual states in seconds.

1. Syntax Overview

  • app "Title Name" - Sets the micro app name
  • author "Your Name" - Sets developer credit
  • set var = "value" - Declares state variable
  • print "log line" - Outputs message to log stream
  • show var - Renders a state variable as a visual box

2. Code Example

app "Health Monitor"
author "Dr. Perseu"

set heart_rate = "72 bpm"
set battery_level = "98%"

print "Syncing sensors..."
print "Telemetry active"

show heart_rate
show battery_level

🚀 Running Spark

Simply create any file with the .spark extension (e.g., todo.spark), write Spark code, and click Run Code (Ctrl+Enter). The output terminal will automatically compile and render your interactive Micro App dashboard!
